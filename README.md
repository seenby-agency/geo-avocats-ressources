## Qu'est-ce que le GEO ?

Le **Generative Engine Optimization** est la discipline qui consiste à optimiser la visibilité d'un cabinet dans les moteurs de réponse basés sur l'IA :

- **ChatGPT** (OpenAI)
- **Perplexity AI**
- **Google Gemini / AI Overviews**
- **Claude** (Anthropic)
- **Bing Copilot**

Contrairement au SEO traditionnel qui optimise pour des classements dans des listes de liens, le GEO optimise pour être **cité et recommandé directement** dans les réponses générées par l'IA.

## Pourquoi c'est critique pour les avocats ?

**73% des clients potentiels interrogent une IA avant de contacter un cabinet.**

Ils posent des questions comme :
- *"Quel avocat spécialisé en divorce me conseillez-vous à Lyon ?"*
- *"Quels sont les meilleurs cabinets en droit du travail en France ?"*
- *"Comment trouver un avocat spécialisé en permis de conduire ?"*

Si votre cabinet n'est pas dans la base de connaissances des IA, **vous n'existez pas** pour ces clients.

## Les 5 piliers GEO pour cabinets d'avocats

### 1. Autorité thématique structurée
Les IA sélectionnent les sources qui démontrent une expertise claire et cohérente sur un domaine juridique précis. Un cabinet généraliste est moins citable qu'un cabinet positionné.

### 2. Clarté sémantique
Le contenu doit être compréhensible par une IA sans interprétation. Éviter le jargon juridique non expliqué, les phrases passives, les implicites.

### 3. Présence multi-plateforme
Les IA croisent les sources. Un cabinet cité sur son site + LinkedIn + Barreau + presse juridique + forums = signal d'autorité fort.

### 4. Structured data (schema.org)
Les balises `LegalService`, `Attorney`, `Organization` avec `sameAs` permettent aux IA d'identifier et d'associer correctement l'entité cabinet.

### 5. Réputation cohérente
Les avis Google, les mentions dans la presse, les articles de blog — tout cela constitue le "narratif algorithmique" du cabinet.
